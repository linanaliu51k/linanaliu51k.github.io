    # koa2

    • express中间件是异步回调，koa2原生支持async/await
    • 新开发框架和系统，都开始基于koa2，例如egg.js
    • express虽然未过时，但是koa2肯定是未来趋势

    # async await 要点:

    • await 后面可以追加 promise对象，获取resolve的值
    • await 必须包裹在 async函数里面
    • async 函数执行返回的也是一个promise对象
    • try-catch 截获 promise 中 reject 的值

    # 安装

    npm install koa-generator -g
koa2 koa2-test
