                    # css居中调研

                    看了近一小天的css居中方案，看到最后有些麻木了。简要记录下各种实现方式

                    # 水平居中

                    • 行内元素水平居中 text-align
                    .header {
  text-align: center;
}
                    行内元素水平居中
                    • 确定宽度的块级元素水平居中 margin: 0 auto
                    .center {
  margin: 0 auto;
}
                    块级元素水平居中
                    • 不确定宽度的块级元素水平居中 (块级变行内) (父: display: table;margin: auto) (display:flex;justify-content: center;) (相对定位与绝对定位) 等.

                    # 垂直居中

                    # 单行内联元素垂直居中 line-height

                    单行内联元素垂直居中

                    # 利用flex布局

                    .box-wrapper {
  display: flex;
  align-items: center;
}

                    /* 对于只有一个元素在容器里面，这种比较简单 */
.box-wrapper {
  display: flex;
}
.box {
  margin: auto;
}
                    flex布局

                    # 利用grid布局

                    .box-wrapper {
  display: grid;
}
.box {
  margin: auto;
}
                    grid布局

                    # 利用表布局(table)

                    .box-wrapper {
  display: table;
}
.box {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}
                    表布局

                    # 利用“精灵元素”

                    .box-wrapper:after {
  content: '';
  display: inline-block; 
  vertical-align: middle; 
  height: 100%;
}
.box {
  display: inline-block; 
  vertical-align: middle;
}
                    精灵元素

                    # 利用transform

                    .box-wrapper {
  position: relative;
}
.box {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
                    transform

                    # 高度固定的子块级元素

                    .box-wrapper {
  position: relative;
}
.box {
  position: absolute;
  top: 50%;
  left: 50%;
  width: 200px;
  height: 50px;
  margin-top: -25px; 
  margin-left: -100px; 
}
                    高度固定的子块级元素 
                    .box-wrapper {
  position: relative;
}
.box {
  position: absolute;
  top: 50%;
  left: 50%;
  top: calc(50% - 25px); 
  left: calc(50% - 100px);
  width: 200px;
  height: 50px;
}
                    高度固定的子块级元素 
                    .box-wrapper {
  position: relative;
}
.box {
  position: absolute;
  left: 0;
  top: 0;
  right: 0;
  bottom: 0;
  margin: auto;
  width: 200px;
  height: 50px;
}
                    高度固定的子块级元素 
                    .box {
  width: 100px;
  height: 50px;
  padding: calc((100px - 50px) / 2);
  background-clip: content-box;
}
                    高度固定的子块级元素

                    # 只针对基于视口的垂直居中, 视口居中也可以使用fixed定位

                    .box {
  width: 100px;
  padding: 10px;
  margin: 50vh auto 0; 
  transform: translateY(-50%)
}
