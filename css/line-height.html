                    # 理解 line-height

                    学习计划开始第三天，今天的目标是理解line-height

                    # 介绍

                    基于该图的说明:
                    • 图中两条红线之间的距离就是行高（line-height),
                    • 上一行的底线和下一行的顶线之间的距离就是行距
                    • 同一行顶线和底线之间的距离是font-size的大小
                    • 半行距 = （line-height - font-size）/2

                    补充说明:

                    • 行高(line-heihgt): 指两行文字间基线之间的距离，用于设置多行元素的空间量。对于块级元素，它指定元素行盒（line boxes）的最小高度，对于非替代的 inline 元素，它用于计算行盒（line box）的高度。
                    • 在inline box模型中,有个line boxes,包裹每行文字。没有height情况下，高度会由一个一个line boxes的高度堆积而成的，line boxes会拿到line-height的值组成高度。
                    • 行高的垂直居中性: line-height的最终表现是通过line boxes实现的，line boxes占据的空间都是与文字内容公用水平中垂线的
                    • 行高的几种表示方法:px/em，或normal，或百分值，或数值，或inherit继承

                    # 4种box

                    • inline box (行内框) 每个行内元素会生成一个行内框(浏览器渲染模型中的一个概念), 行内框的高度等于font-size，设定line-height时，行内框的高度不变，改变的是行距
                    • line box（行框）行框(浏览器渲染模型中的一个概念)是指本行的一个虚拟的矩形框，由该行中行内框组成, 行框高度等于本行中所有行内框高度的最大值.
                    • content area（内容区）: 围绕着文字的一种box, 其高度取决于font-size和padding
                    • containing box 是包裹着上述几种的最外层box。

                    # 各种取值说明

                    • normal: (默认) 桌面浏览器使用默认值，约为1.2, 这取决于元素的 font-family.
                    • 数字(number): 计算值是无单位数字乘以该元素的字体大小, 继承时继承的是该数字，之后再结合字体大小进行计算.（推荐方式）
                    • 长度(length): 设置固定的行间距, 长度单位都可使用。
                    • 百分比(%): 计算值是给定的百分比值乘以元素计算出的字体大小(继承时继承的是line-height本身计算后得到的值)
                    • inherit: 规定应该从父元素继承 line-height 属性的值

                    # 继承说明

                    • inherit: 继承父元素line-height的值，所以父元素的是多少就是多少
                    • length: 与font-size无关，不会随着 font-size 做相应比例的缩放, 原来是多少，继承多少。
                    • 百分比: line-height 是根据自身的 font-size 计算出来的, 子元素会继承父元素的line-height计算后的最终值 例如 16 * 120% = 19.2px，继承的是19.2px.
                    • normal: 行高取决于浏览器的解析，一般是1.2.其子元素不再继承其line-height计算后的最终值，而是根据子元素自身的 font-size 进行计算 。即默认的1.2 乘以自身的font-size值
                    • number: 比normal灵活，设置一个自定义的值，继承的是数字，最终由该数字乘以自身font-size
