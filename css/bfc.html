                    # bfc 学习

                    学习计划开始第二天，今天的目标是学习bfc, 但是又顺便看了不少其他东西，东西是真的多啊。看的眼花缭乱，也来不及细细看书。

                    # 介绍

                    • Box: CSS布局的基本单位, 不同类型的Box，会参与不同的 Formatting Context。CSS2.1 中只有 BFC 和 IFC, CSS3 中还增加了 GFC 和 FFC。
                    • 格式化上下文指的是初始元素定义的环境. 它是页面中的一块渲染区域，有一套渲染规则，决定了其子元素如何布局，以及和其他元素之间的关系和作用
                    • 常见定位方案有普通流，浮动，绝对定位等。
                    • 流体特性: 块状水平元素, 在默认情况下（非浮动、绝对定位等），水平方向会自动填满外部的容器；如果有margin-left/margin-right, padding-left/padding-right, border-left-width/border-right-width等，实际内容区域会响应变窄

                    # BFC

                    • Formatting context(格式化上下文) 是 W3C CSS2.1 规范中的一个概念。它是页面中的一块渲染区域，并且有一套渲染规则，它决定了其子元素将如何定位，以及和其他元素的关系和相互作用。属于定位方案的普通流
                    • 具有 BFC 特性的元素可以看作是隔离了的独立容器，容器里面的元素不会在布局上影响到外面的元素，并且 BFC 具有普通容器所没有的一些特性

                    # 创建格式化上下文(常见)

                    • 根元素或包含根元素的元素
                    • 浮动元素（元素的 float 不是 none）
                    • 绝对定位元素（元素的 position 为 absolute 或 fixed）
                    • display 为 inline-block、table-cells、flex
                    • overflow 除了 visible 以外的值 (hidden、auto、scroll)
                    • display 值为 flow-root 的元素(无副作用)

                    # BFC布局规则

                    • 内部的Box会在垂直方向，一个接一个地放置
                    • Box垂直方向的距离由margin决定。属于同一个BFC的两个相邻Box的margin会发生重叠
                    • 每个元素的margin box的左边， 与包含块border box的左边相接触(对于从左往右的格式化，否则相反)。即使存在浮动也是如此
                    • BFC的区域不会与float box重叠
                    • BFC就是页面上的一个隔离的独立容器，容器里面的子元素不会影响到外面的元素。反之也如此
                    • 计算BFC的高度时，浮动元素也参与计算

                    # BFC常见应用(很好的体现了上面的BFC布局规则)

                    • 自适应两栏布局(可以参考张鑫旭的文章)
                    .box1 {
  float: left;
  width: 20px;
  height: 20px;
  border: 1px solid blue;
}
.box2 {
  display: table-cell;
  width: 9999px;
}
                    自适应两栏布局
                    • 可以阻止元素被浮动元素覆盖
                    .left {
  float: left;
  width: 20px;
  height: 20px;
  border: 1px solid blue;
}
                    浮动元素被覆盖，同时文字环绕 
                    .right {
  overflow: hidden;
}
                    产生bfc, 浮动元素不被覆盖
                    • 可以包含浮动元素——清除内部浮动
                    .float {
  float: left;
  width: 40px;
  height: 40px;
  border: 1px solid blue;
}

                    .parent2 {
  overflow: hidden;
}
                    • 分属于不同的BFC时可以阻止margin重叠
                    <div class="wrapper2">
  <div class="child"></div>
  <div class="child"></div>
</div>

                    .child {
  width: 20px;
  height: 20px;
  margin: 10px 0;
  border: 1px solid blue;
}

                    <div class="wrapper2">
  <div class="bfc">
    <div class="child"></div>
  </div>
  <div class="bfc">
    <div class="child"></div>
  </div>
</div>

                    .bfc {
  overflow: hidden;
}
